• 상단에 터미널에서 새 터미널 클릭하고 열린 터미널에
  • git init 이라고 입력하면 깃이 내 코드를 기록? 감시함. (예 : git init)
  • git add 파일명 적고 (예 : git add index.html )
  • git commit -m '메모' 라고 적으면 그 순간이 기록됨 (에 : git commit -m '깃 테스트중')
  • 또 새롭게 저장하고 싶은 순간이 생기면 git add 부분부터 재시작
  • (git add index.html style.css)로 두개 동시 기록
  • git status로 상태창열기 가능
  • commit 내역 조회 (git log --all --oneline)
